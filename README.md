webtul
====================
A set of web developing tools instead of framework, and tul sounds like tool.
Up to date version: 0.31

installation
--------------------
    wget https://github.com/zagfai/webtul/archive/v0.31.zip
    unzip v0.31
    cd webtul-0.31 && sudo python setup.py install

modules
--------------------
* db
* log
* utils
